`Filterkontrolle`
==================

A water filtration control system written for the ATmega328 Microcontroller in Rust.



Features:

- Bluethooth communication with an App 

- Waterlevel measurement using an ultrasonic sensor

- Leak detection

- Different filter jobs

## Usage

If you don't have them already, install [`cargo-generate`] and [`ravedude`]:

```bash
cargo install cargo-generate
cargo install ravedude
```

Then upload the program:

```bash
cargo run
```
